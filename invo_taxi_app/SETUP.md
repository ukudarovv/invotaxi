# Инструкция по установке и запуску проекта Инво Такси

## Требования
- Flutter SDK (версия 3.10.1 или выше)
- Dart SDK (входит в состав Flutter)

## Установка Flutter (если не установлен)

### Windows:
1. Скачайте Flutter SDK с https://docs.flutter.dev/get-started/install/windows
2. Распакуйте архив в удобное место (например, `C:\src\flutter`)
3. Добавьте Flutter в PATH:
   - Откройте "Переменные среды"
   - Добавьте путь к `flutter\bin` в переменную PATH
4. Перезапустите терминал

### Проверка установки:
```bash
flutter doctor
```

## Запуск проекта

1. Перейдите в директорию проекта:
```bash
cd invo_taxi_app
```

2. Установите зависимости:
```bash
flutter pub get
```

3. Запустите приложение:
```bash
# Для запуска на подключенном устройстве/эмуляторе
flutter run

# Или укажите конкретное устройство
flutter devices  # посмотреть доступные устройства
flutter run -d <device_id>

# Для запуска в веб-браузере
flutter run -d chrome

# Для запуска на Windows
flutter run -d windows
```

## Мокап данные

Проект использует мокап данные из `lib/data/mock_db.dart`:

- **5 регионов**: Северный, Южный, Центральный, Восточный, Западный
- **10 водителей**: первые 5 онлайн, с разными моделями машин
- **51 пассажир**: включая тестового пассажира (Тестов Тест Тестович, +7 (777) 777-77-77)
- **Заказы**: создаются через UI

## Тестовые данные для входа

### Пассажир:
- Телефон: `+7 (777) 777-77-77` или `7777777777`

### Водитель:
- Используйте любой телефон из списка водителей (например, `+7 (900) 100-10-10`)
